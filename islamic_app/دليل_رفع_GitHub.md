# 📱 دليل بناء APK مجاناً - خطوة بخطوة

## ✅ المطلوب منك فقط:
- متصفح إنترنت (Chrome أو Firefox)
- حساب GitHub مجاني
- الملفات التي حملتها (islamic_app_flutter.zip)

---

## 🔷 الخطوة 1: إنشاء حساب GitHub

1. افتح المتصفح وادخل: **github.com**
2. اضغط **Sign up**
3. أدخل الإيميل، كلمة مرور، واسم المستخدم
4. فعّل الحساب من الإيميل

---

## 🔷 الخطوة 2: إنشاء مستودع جديد

1. بعد تسجيل الدخول اضغط زر **"+"** في الأعلى
2. اختر **"New repository"**
3. اكتب الاسم: `islamic-app`
4. اختر **Public**
5. ✅ ضع علامة على **"Add a README file"**
6. اضغط **"Create repository"**

---

## 🔷 الخطوة 3: رفع الملفات

### طريقة سهلة (Drag & Drop):

1. افتح الـ ZIP وفك ضغطه على سطح المكتب
2. في GitHub، اضغط **"uploading an existing file"**
3. **اسحب** مجلد `islamic_app` بالكامل إلى الصفحة
4. في أسفل الصفحة اضغط **"Commit changes"**

### أو ارفع الملفات واحداً واحداً:
- اضغط **Add file** → **Upload files**
- ارفع كل الملفات

---

## 🔷 الخطوة 4: تشغيل البناء التلقائي

بمجرد رفع الملفات، GitHub Actions **سيبني APK تلقائياً!**

### لمتابعة البناء:
1. اضغط على تبويب **"Actions"** في مستودعك
2. ستجد عملية بناء تعمل (دائرة صفراء 🟡)
3. انتظر 5-10 دقائق حتى تكتمل (دائرة خضراء ✅)

---

## 🔷 الخطوة 5: تحميل APK

### بعد اكتمال البناء:

**طريقة 1 - من Releases:**
1. اضغط على **"Releases"** في الصفحة الرئيسية
2. ستجد `نور الإسلام v1.0.x`
3. حمّل `app-release.apk` ✅

**طريقة 2 - من Artifacts:**
1. في تبويب Actions، اضغط على البناء المكتمل
2. انزل للأسفل حتى قسم **"Artifacts"**
3. حمّل `islamic-app-apk`

---

## 📲 تثبيت APK على الأندرويد

1. انقل الـ APK للهاتف (بلوتوث أو كابل USB)
2. في الهاتف: **الإعدادات** → **الأمان** → فعّل **"مصادر غير معروفة"**
3. افتح ملف الـ APK وثبّته ✅

---

## ❓ مشاكل شائعة وحلولها

| المشكلة | الحل |
|---------|------|
| البناء فشل (❌ أحمر) | اضغط على البناء واقرأ الخطأ، ثم أخبرني |
| لا أجد Releases | ابحث في Actions → Artifacts |
| APK لا يُثبّت | فعّل "مصادر غير معروفة" في إعدادات الهاتف |
| الملفات كثيرة | ارفع الـ ZIP كاملاً ثم استخدم GitHub CLI |
